# Pattern Generation (C)

Console pattern programs using loops and conditionals.

## Programs

- `pattern_diamond.c` — diamond pattern using `*`
- `pattern_hollow_square.c` — hollow square pattern using `*`
- `pattern_number_triangle.c` — number triangle pattern

## Build & Run (PowerShell)

```powershell
cd "main projects/patterns"
gcc -std=c11 -Wall -Wextra -o pattern_diamond.exe pattern_diamond.c
./pattern_diamond.exe
```
